# GitBoti

Console app to perform git actions.

## Perform the following actions:
  - Basic actions:
    - status
    - add *
    - commit -m
    - push
  - New Repo
    - clone
  - Ramas
    - Listar las ramas locales
    - Listas las ramas remotas
    - Crear una nueva rama
    - Cambiar de rama
